# AVODOD 🏛️

**AVODOD** — bu ajdodlarimizning bebaho ilmiy merosini zamonaviy texnologiyalar yordamida bugungi avlodga yetkazuvchi "raqamli ko‘prik". Loyiha nomi **Ajdod** va **Avlod** so‘zlarining birlashmasidan hosil bo‘lgan bo‘lib, ilmning asrlar osha davomiyligini anglatadi.

## 🌟 Loyiha falsafasi
Ilm — bu uzilmas zanjir. Bizning maqsadimiz buyuk olimlarimiz (Xorazmiy, Beruniy, Ibn Sino va b.) tomonidan yaratilgan murakkab qo‘lyozmalarni bugungi dasturchilar, talabalar va ilmga qiziquvchilar uchun tushunarli bo‘lgan interfaol formatga o‘tkazishdir.

## 🏗️ "Ko‘prik" konsepsiyasi
Ilova interfeysi uchta asosiy qatlamdan iborat bo‘ladi:

1.  **Arabcha matn:** Qo‘lyozmaning skanerlangan varianti yoki asl arabcha matni.
2.  **Tarjima:** O‘zbek tilidagi so‘zma-so‘z va aniq ilmiy tarjima.
3.  **Izoh:** Matndagi formulalarning zamonaviy dasturlash tillari (Python/JS) va vizual tushuntirishlar bilan tahlili.
